# Add cover to `_quarto.yml`

Add cover to `_quarto.yml`

## Usage

``` r
add_cover(report_path = ".", cover_pdf)
```

## Arguments

- report_path:

  The path to the folder containing the report. Defaults to the current
  working directory.

- cover_pdf:

  The path to a PDF file. The first page of this PDF file will be used
  as the cover of the report.
